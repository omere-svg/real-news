# Reserve Bank of Australia  (SourceId: oceania-rba)

- **Tier:** A — primary-official (central bank publishing its own authoritative macro/FX statistical data)
- **Role:** SIGNAL (numeric scoring context — interest rates, AUD exchange rates, money-market data; no prose content records emitted)
- **Endpoint probed:** GET `https://www.rba.gov.au/statistics/tables/csv/f11.1-data.csv`
- **Format:** CSV (proprietary RBA layout: 8-row metadata header then date-keyed daily observation rows; no JSON or SDMX offered)
- **Auth:** none   |   **Rate limit:** not documented; no rate-limit headers observed; single CSV file (~125 KB) returned in ~0.78 s; recommend polling at most once daily (data updates once per business day)
- **Probe status:** LIVE-CONFIRMED (HTTP 200, `Content-Type: application/octet-stream`, 128 519 bytes, 23 currency-pair columns plus Trade-Weighted Index, Series IDs confirmed in row 9)
- **Region mapping:** asserts Oceania (AUD-denominated; covers AUD/USD, AUD/NZD, AUD/CNY, AUD/JPY and 19 other pairs)
- **Topic mapping:** Exchange Rates / Interest Rates / Central Bank Policy -> Business (primary); Geopolitics (secondary, for AUD/CNY and AUD/JPY stress signals)
- **Signals yielded:** points? NO   mentions? NO   tone? NO — yields numeric float exchange-rate observations per calendar date; signal fields are the per-pair rate columns (e.g. row `17-Jun-2026`, column `A$1=USD` = `0.6427`); no engagement metadata
- **externalId (dedup key):** composite of table ID + Series ID + date — e.g. `F11.1/FXRUSD/17-Jun-2026`; Series IDs are in row 9 of the CSV (`FXRUSD`, `FXRTWI`, `FXRCR`, `FXRJY`, `FXREUR`, `FXRSKW`, etc.)

- **Storage/ToS note:** RBA data is licensed under Creative Commons Attribution 4.0 International (CC BY 4.0). Server-side caching and redistribution are permitted provided attribution reads "Source: Reserve Bank of Australia [year]" or "Source: RBA [year]". Third-party-sourced columns (WM/Reuters for AUD/USD; IMF for SDR) cannot be redistributed without the originator's permission — strip or flag those two columns if redistributing. No explicit prohibition on automated polling found.
- **Verdict:** TRIAL — adds the **AUD official central-bank FX/rate axis for Oceania**: the only source in the current set (HN/arXiv/GDELT/Knesset/SEC/Wikipedia) anchored to AUD, providing authoritative daily AUD exchange rates and the Australian cash rate target usable as economic-stress context for Business and Geopolitics stories in the Oceania bloc. Downgraded from ADOPT because (a) RBNZ is geo-blocked so NZ coverage is absent, (b) the CSV-only format requires custom parsing (no SDMX/JSON like ECB-SDW), and (c) purely numeric — no prose content, so it only enriches existing items rather than sourcing new ones.
- **Risks:** (1) No public API contract or SLA — RBA could change CSV column order or filename conventions without notice; pin to named Series IDs (row 9), not column positions. (2) WM/Reuters (AUD/USD) and IMF (SDR) columns carry third-party licensing — do not cache or expose those two series externally. (3) RBNZ website consistently returns HTTP 403 (geo-block or bot-detection), so the "RBNZ" portion of the proposed source is unavailable; this sheet covers RBA only. (4) Data updates once per business day (after ~4 pm AEST); stale reads outside that window are expected and normal. (5) CSV metadata header is 8 rows deep — parsers must skip/handle before reaching observations.
